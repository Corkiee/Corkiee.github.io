What I've learned so far.

Posted on 12/13/2019

So far I'm really enjoying this coding stuff. Khan Academy is really useful for deep-dives and getting more knowledge on the subject. Before Khan Academy, I was using FreeCodeCamp, which had little user interaction. It usually just gave me a couple of lessons and then left me on my own to do a project without any handholding. I was fine with that at first, thinking it would strengthen my coding abilities, however I soon realised I was relying on others' completed projects for the code, and I didn't reall understand what I was doing. On Khan, it forces you to learn, and I like that a lot.

Anyway! This week in my coding adventures, I started out learning about Binary Algorithms and finished it strong with learning more about CSS selectors. CSS is such an interesting way to style pages that I want to learn every little bit of it. It might be my favorite. There's just so many things you can do with a tool like this. From pushing all of the text to the right, to wrapping text around a framed image, just like in a newspaper.
